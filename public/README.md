Place the hero image you want the site to use at `public/lifecare-hero.jpg`.

- File name: lifecare-hero.jpg
- Recommended size: 1600×1200 (or similar 4:3/3:2 ratio)
- Recommended format: JPEG or WebP

After placing the file, rebuild the site (or restart the dev server) to see the new hero image.

Commands:

```bash
npm run dev   # local dev server
npm run build # production build
```
